Empathize: 
  Help users that struggle staying consistent with their skincare routine 
  Target audience: users 18-30 interested in skincare

Define: 
  Problem statement 
    Busy people need a simple way to remember and track their routines because forgetting steps or being inconsistent makes it hard to see real results
  User app needs 
    Remember which products to use and in what order
    Separate morning and night routines
    Remind users to do their skincare routine at certain times 
    App should show history 
    Should tell users what skincare can and can’t be combined

  Prioritize

Prototype: 

  Pages 
  Home / today screen 
    Displays today’s date
    Shows Morning Routine section (with progress bar)
    Shows Night Routine section (with progress bar)
    Quick buttons to “Go to Morning Routine” and “Go to Night Routine”
  Routine detail screen 
    List of all steps in the current morning OR night routine
    Reorderable list (drag & drop)
    Checkboxes/toggles to mark completion
    List of all user products used in that routine
    Tapping a product → opens Product Detail Screen	
  Add / edit screen
    Add and edit skincare routine 
  Skin concerns 
    User selects from a list of skin concerns
    Suggested skincare products based on concern 
    Shows suggested skincare focus
  Product detail screen 
    product name and category (cleanser, serum, … )
    Key ingredients 
    What skincare product work well and should not be mixed with this product 
    
  Functionalities 
    Delete routine steps 
    Reorder steps 
    Create routine steps 
    Edit routine steps (edit name, type, details) 
    Check off products / days 
    Saving data 
    Add and view product details 
    Select skincare concerns 
    Suggested skincare focus based on skin concerns 
    View product incompatibilities
  
  Advanced features 
    Using gestures
    Simple animations
  
  Design requirements 
    Visibility of system status
    Consistency and standards
    Error prevention
  
  Human interface guidelines
